### Phase 1: The Master Architecture & Theme Prompt

*Feed this first to establish the global rules, colors, and the precise cinematic aesthetic.*

> **Role:** You are a visionary Creative Developer and UX/UI Architect. We are building a React + Vite + Tailwind CSS application for a hackathon.
> **Core Objective:** Build a GenAI Smart Stadium platform for the 2026 World Cup. The UI must be breathtaking, cinematic, and entirely unique.
> **Global Aesthetic Directives (The "Golden Era Cinematic" Look):**
> * **Color Palette:** Base the UI on deep, moody tones.
> * `Backgrounds`: Obsidian black (`#0A0A0A`) with subtle radial gradients of rich emerald pitch green (`#0F291E`).
> * `Accents`: Metallic vintage gold (`#D4AF37`) and brushed brass.
> * `Text`: Off-white/parchment (`#F3EFE7`) for reduced eye strain and a premium vintage feel.
> 
> 
> * **Textures & Lighting:** The UI must feel like a 90s sports cinema masterpiece. Implement a global, subtle SVG film grain overlay on the background. Use CSS box-shadows to create warm, volumetric glowing effects (warm tungsten lighting) behind active components, mimicking stadium floodlights in the evening.
> * **Typography:** Primary headings must use a heavy, dramatic serif font (like *Playfair Display* or *Cinzel*) to evoke historic matchday posters. Body text must be a crisp, modern sans-serif (like *Inter* or *Space Grotesque*) for maximum readability of GenAI data.
> 
> 
> **Action:** Generate the `tailwind.config.js` with these custom colors, fonts, and custom glow utilities. Then, generate the global `App.jsx` structure with the overarching film-grain texture overlay and routing setup.

### Phase 2: The Bombastic Hero Section Prompt

*This handles the crucial first impression.*

> **Task:** Create the `CinematicHero.jsx` component using React, Tailwind CSS, and Framer Motion. This is the landing page hook.
> **Visual & Motion Requirements:**
> 1. **Background:** A full-screen `<video>` element set to autoplay, loop, and mute. Overlay it with a dark, warm-toned vignette gradient (darker at the edges, slightly transparent in the center) so text pops.
> 2. **Typography Animation:** The main headline (e.g., "Symphony of the Pitch. Powered by GenAI.") must stagger in letter-by-letter using Framer Motion, starting from a blur state and resolving into sharp focus, mimicking a camera pulling focus.
> 3. **The Call to Action (CTA):** A massive, magnetic button labeled "Initialize Stadium GenAI". Instead of a standard flat button, make it look like a vintage mechanical switch with modern glassmorphism. When hovered, it should emit a warm, golden glow (`box-shadow`) and scale up slightly.
> 4. **Scroll Indicator:** A custom, glowing gold chevron at the bottom, bouncing softly, urging the user to discover the features below.
> 
> 
> **Action:** Write the complete, production-ready code for `CinematicHero.jsx`, ensuring perfect responsiveness and high-end motion design.
